array.h File Reference

Detailed Description

Array

Unlike tuple(see tuple.h), array is variable size of a continuous memory. see voba_array_s

Many APIs modify the first argument in place with side effect. It will destory the first argument, so use the return value instead. for example

voba_value_t a = voba_array_from_tuple(tuple);
a = voba_array_push(a,xx);
