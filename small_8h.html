#define VOBA_NIL   0

constant value nil 

+---------------------------+------------------------+
|                    0                               |
+---------------------------+------------------------+

This class only contain one value, VOBA_NIL. The least significant 3-bits is 000, which is as same as VOBA_TYPE_FUNC, but no function address is zero, so it is safe to share the same type tag with VOBA_TYPE_FUNC.

VOBA_NIL is usually represents

  • end of a list
  • empty value
  • it has the special value of zero.
