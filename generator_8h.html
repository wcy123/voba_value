INLINE voba_value_t voba_make_generator ( voba_func_t  f,
voba_value_t  self,
voba_value_t  args 
)
Returns
a callable object which capture a function pointer f, self and args.

When the object is callled via voba_apply, cg_invoke is used to start the generator, i.e. start f(self,args). In the function f, cg_yield could be invoked to suspend the generator.
