Detailed Description

Tuple

Tuple is a fixed size array.

+------------------------------------------------+-------+
| data                                           | type1 |
+------------------------------------------------+-------+
| .    value (61 bits)                           |  010  |
+-|----------------------------------------------+-------+
  |
  |       +----------------+------------+
  `---->  |         size(64bits)        | 
          +----------------+------------+
          |         voba_value_t a1     |
          +-----------------------------+
          |         voba_value_t a2     |
          +-----------------------------+
          |            ....             |
          +-----------------------------+

It is usually used for function call as below

voba_value_t args[] = {2, arg1, arg2};
voba_value_t a_tuple = voba_make_tuple(args);
