Detailed Description

hashtable

A hashtable is a user define class.

+------------------------------------------------+-------+
| data                                           | type1 |
+------------------------------------------------+-------+
| .    value (61 bits)                           |  101  |
+-|----------------------------------------------+-------+
  |
  |     +------------------------------+--------------------+
  `->   |           voba_cls_hashtable                      |
        +------------------------------+--------------------+
        |           hash_table_c ...                        |
        +------------------------------+--------------------+

voba_hash_insert and voba_hash_find return a pair(see pair.h), DOT NOT modify the head of the pair, otherwise, the integrity of the whole hash table is destroyed.

most of them are implemented in c++, using vhash, see http://github.com/wcy123/vhash
