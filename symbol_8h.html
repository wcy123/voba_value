Detailed Description

symbol 

+------------------------------------------------+-------+
| data                                           | type1 |
+------------------------------------------------+-------+
| .    value (61 bits)                           |  010  |
+-|----------------------------------------------+-------+
  |
  `-->  +----------------------------+------------------------+
        |           symbol name ( a string )                  |
        +----------------------------+------------------------+
        |           symbol value (any voba value)             |
        +----------------------------+------------------------+

symbol is similar to pair. A interned symbol is inserted into a symbol table. When looking up a symbol with a same name, we always get the same symbol object. One symbol could be interned into more than one symbol table.

see symbol_table.h
