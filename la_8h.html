Detailed Description

listarray

In LISP, we often have the push and reverse idiom, list array is the optimization (pre-mature, sorry) for it. Using array to represent list is more space efficient.

interface:

  • la_is_nil(LA)
  • la_car(LA)
  • la_cdr(LA)

for example:

voba_value_t x = voba_make_array_0();
voba_value_t cur = voba_la_from_array0(x);
while(!voba_la_is_nil(cur)){
voba_value_t a = voba_la_car(cur);
// do something with \a a
cur = voba_la_cdr(cur);
}
