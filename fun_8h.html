Detailed Description

Basic type of function object

A function object is a function pointer, the assocated 3-bit type tag is 000.

+--------------------+------------------------+
| .     61 bits value                   | 000 |
+-|------------------+------------------------+
  |
  `----> address of a function.

The type of function pointer is typedef as voba_func_t

typedef voba_value_t (*voba_func_t)(voba_value_t self, voba_value_t args);

The function must be 16 aligned, usually use VOBA_FUNC For example:

VOBA_FUNC static voba_value_t a_function(voba_value_t self, voba_value_t args)
{
.......
}

For an s-exp,

1 (f a1 a2 a3 ...)

where self is f and args is a tuple [ a1, a2, a4 , .... ]

f could be any callable object, like, a closure, a generic function, a function, an tuple, a hash table, or any other user defined objects whose class method implements the voba_gf_apply
